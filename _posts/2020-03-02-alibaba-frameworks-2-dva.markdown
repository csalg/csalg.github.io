### Setup

Setup is fairly straight forward:

```
npm install dva-cli -g
```

We can check nothing blew up:

```
dva -v
# dva-cli version 0.9.1
```

### New Project

```
dva new dva-quickstart; cd $_
npm start
```

And we should see a development server start.
